## What This Is

Shiro's Scanner is a browser-oriented scanner interface made for FiveM-related use. It uses HTML as its foundation, giving you a way to work with scanner-style functionality through a regular web browser instead of a traditional desktop app.

The project is organized as a compact website that can be opened directly in a browser or hosted like any other static page. Because its focus is tied to FiveM, future changes are expected to stay compatible with that environment and with any interface refinements needed for a smoother web experience.
